# 核心命令 <Badge type="danger" text="开发中" vertical="top"/>

使用和控制 RinBot 的一些基本命令。

:::warning
这个模块无法被关闭
:::

## 帮助

获取帮助信息，导航至[本网站](../README.md)

### 命令格式

```:no-line-numbers
/help
```

## Ping

检查 RinBot 是否在线，回复一些神秘的话

### 命令格式

```:no-line-numbers
/ping
```

## 查看用户信息

查看当前 IM 平台下用户账号所对应的用户信息

### 命令格式

```:no-line-numbers
/info
/信息
```

## 查看 RinBot 运行状态

获取当前 RinBot 运行的环境以及状态

### 命令格式

```:no-line-numbers
/status
```
