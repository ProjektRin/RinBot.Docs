# 关于 RinBot

RinBot 为~~超级扇贝王~~ [AkulaKirov](https://github.com/AkulaKirov) 开发的一款 IM 机器人，设计初衷是为了让所有人都能看上色图。

在运营过程中听取反馈和不断改进，才诞生了现在的铃酱。
