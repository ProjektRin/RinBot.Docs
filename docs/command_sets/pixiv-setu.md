# Pixiv色图

来点色图，懂得都懂。

本功能使用了 [Lolicon API](https://api.lolicon.app)。

:::warning
这个命令集所包含的元素可能 **并不适合所有人**。
:::

## 获取色图

获取指定张包含指定标签的色图

### 命令格式

```:no-line-numbers
/setu [<tags>] [-n <num>] [-r18]
/色图 [<tags>] [-n <num>] [-r18]
```

| 参数 |  名称  |  类型  |  描述  |  必要  |
|:----:|:----:|:----:|:----:|:----:|
| tags |  标签  |  List\<string\>  |  要包含的标签，多个标签用空格分隔开，最多 3 个  |  否  |
| -n \<num\> |  数量  |  int  | 要获取色图的数量，默认为 1，最大为 100，实际返回数量可能会根据搜索条件而变化  |  否  |
| -r18 |  R18 模式  |  bool  |  开启R18模式，慎用  |  否  |
